
## Descarga

- Descargar imagen del registry.

```bash
docker pull docker.swagger.io/swaggerapi/swagger-editor
```

## Ejecución

- Ejecutar Swagger utilizando contenedor en Docker.

```bash
docker run -d -p 80:8080 docker.swagger.io/swaggerapi/swagger-editor
```
